🔑 Token do GitHub

Para publicar diretamente, você precisa de um Personal Access Token com permissão Contents: Read & Write no repositório docesflor/admin. O token fica salvo apenas no seu navegador (localStorage).

Como criar o token

  1. Acesse github.com/settings/tokens
  2. Clique em Generate new token (fine-grained)
  3. Em Repository access: selecione Only select repositories → docesflor/admin
  4. Em Permissions → Repository → Contents: Read and write
  5. Clique Generate token, copie e cole acima
